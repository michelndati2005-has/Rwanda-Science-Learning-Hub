About Our School

A Legacy of Faith & Excellence

Saint François de Sales School is a distinguished Catholic primary school dedicated to the holistic development of every child. Founded on the teachings of Saint François de Sales — the patron saint of writers, educators and the deaf — our school fosters intellectual curiosity, spiritual growth and moral integrity.

🏆 Our Mission

To provide an excellent Catholic education that forms the whole person — intellectually, spiritually, morally and socially — so that each student becomes a responsible and compassionate citizen.

📋 Role Capabilities
👔 Director: Full super-admin control. Add/remove all users, view all reports, generate report cards, manage website, view activity log.
🏫 Head Teacher: Full portal access. Manage teachers, students, classes, quiz, proclamations, announcements, and public website.
👩‍🏫 Teacher: Manage own class — upload notes, set homework, record attendance, enter grades, generate report cards, message parents/students.
👨‍🎓 Student: View notes, homework, grades, report card, timetable, take quiz, send messages. Use 🔊 Listen on any note for audio reading.
👪 Parent: View child's grades, attendance, homework, report card, timetable, announcements, and message the teacher.
🔊 Audio Learning (Text-to-Speech)
Every note, homework and classwork card has a 🔊 Listen button. Click it to hear the content read aloud.
• Click again to stop the audio at any time.
• Click the 🔊 icon in the top navigation bar to adjust reading speed, pitch, and voice.
• Works in Chrome, Edge, and Safari. Best experience with Chrome.
• Report cards can also be read aloud using the 🔊 Read Report Aloud button.
📄 Report Cards
Generating a report card: Go to Report Cards in the sidebar → Select class and student → Choose term → Add a teacher comment → Click Generate Report Card.

Exporting:
• 🖨️ Print / Save PDF — opens a clean print window. Use your browser's "Save as PDF" option.
• 📝 Export Word — downloads a .doc file you can open in Microsoft Word.
• 🔊 Read Aloud — the system will read the report card summary aloud.
📊 Entering Marks
Manual entry: Go to Grades → Click any mark cell to edit it → Averages update automatically.

Excel upload: Click 📤 Upload Excel → Download the blank template first → Fill in marks → Upload the completed file.
The template format is: Student Name | Subject Test | Subject Exam
💬 Messaging
Click Messages in the sidebar to open the messaging centre.
• Click any conversation thread on the left to open it.
• Type your message and click Send Message.
• Click + New to start a new conversation.
• All messages between teachers, students, and parents are kept here.
🌍 Language / Langue
Click the 🌍 Français / English button in the top navigation bar to switch the public website interface language between English and French.
